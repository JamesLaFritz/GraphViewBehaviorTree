    Class BehaviorTreeEditor

    Derive from UnityEditor.EditorWindow class to create an editor window to Edit Behavior Tree Scriptable Objects. Requires file named "BehaviorTreeEditor.uxml" to be in an Editor Resources Folder Uses Visual Elements requires a BehaviorTreeView an an UnityEngine.UIElements.IMGUIContainer with a name of InspectorView.
